Swipe ID

 

Use this screen to enter the ID number when accepting a check as tender or if the transaction involves PAT cash.

 

What Can I Do On This Screen?

 

Swipe the ID or enter the identification number in the ID Number field, then press Next.

 

Press Undo to return to the previous screen.

 

Press Cancel to stop the tendering function and return to the Sell Item screen.

 

Press Delete to clear the current field entry.
