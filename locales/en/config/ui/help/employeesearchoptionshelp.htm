Employee Search Options

 

Use this screen to choose a search option to find information about an employee who is authorized to use the system.

 

What Can I Do On This Screen?

 

Choose an option from the navigation bar on the right side of the screen:

 

Emp. ID

Choose this option to search for an employee based on login ID.

 

Emp. Name

Choose this option to search for an employee based on first and last name.

 

Role

 

Choose this option to search for an employee based on role or level of access in the system.

 

 

Press Undo to return to the previous screen.

 

Press Cancel to exit the function.
