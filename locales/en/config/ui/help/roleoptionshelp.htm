Role Options

 

Use this screen to choose the role function you want to perform. You can look up a role and/or security access as well as add a new role to the system.

 

What Can I Do On This Screen?

 

Choose an option from the navigation bar on the right side of the screen:

 

Find

Choose this option to find information about a role.

 

Add

Choose this option to add a role to the system.

 

 

Press Undo to return to the previous screen.

 

Press Cancel to exit the function.
