Gift Certificate Tender

To tender a transaction using a Gift Certificate, type the amount of the gift certificate if different from the balance due and press F7/Gift Cert..  When the Cert Entry screen appears, perform these steps: 

·         Scan the barcode, or type the gift certificate number (minimum 12 digits) and press Enter/Next.

·         The system checks the database to approve the validity of the certificate based on dollar amount, expiration date, and certificate number.

·         If the gift certificate tender is greater than the balance due:

–                    The system evaluates the difference against the Max % Cash Change For Gift Cert parameter.

–                    If the difference is greater than the maximum, the Over Max Chg screen appears.  Press Enter/Next to return to the Tender Options screen to select a different tender option.

·         If the amount of the gift certificate is less than the balance due, you return to the Tender Options screen to select an additional tender type to complete the transaction.  The gift certificate tender item is displayed.

·         The gift certificate tender is indicated on the transaction receipt, and you are returned to the Sell Item.
