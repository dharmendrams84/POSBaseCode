House Account Options

 

Use this screen to choose the house account function you want to perform.

 

What Can I Do On This Screen?

 

Choose an option from the navigation bar on the right side of the screen:

 

Inquiry

Choose this option to find information about a house account.

 

Enrollment

Choose this option to enroll a customer in a house account.

 

Temporary Pass

Choose this option to issue a temporary pass to a customer.

 

Reference Number Inquiry

Choose this option to enter a reference number to have an enrollment application approved.

 

Payment

Choose this option to issue a payment from a house account.

 

 

Press Undo to return to the previous screen.
