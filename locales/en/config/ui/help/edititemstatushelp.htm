Edit Item Status

 

Use this screen to change the item status.

 

What Can I Do On This Screen?

  

Select the item(s), change the status, then press Next.

 

 Choose an option from the navigation bar on the right side of the screen:

 

Filled

Choose this option to change the status to filled.

 

Pending

 

Choose this option to change the status to pending.

Pick Up

 

Choose this option to change the status to pick up.

Canceled

Choose this option to change the status to canceled.

 

 

 

Press Undo to return to the previous screen.

 

Press Cancel to exit the function.
