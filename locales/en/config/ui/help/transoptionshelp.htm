Transaction Options

 

Use this screen to choose a transaction option.

 

What Can I Do On This Screen?

 

Choose an option from the navigation bar on the right side of the screen:

 

Tax

Choose this option to perform tax functions. This option is not available when VAT is enabled.

 

Sales Associate

 

Choose this option to link a sales associate.

Gift Registry

 

Choose this option to link a gift registry.

Gift Receipt

 

Choose this option to print a gift receipt.

Layaway

 

Choose this option to perform a layaway.

Send

 

Choose this option to send items to another address.

Post Void

 

Choose this option to void a posted transaction.

Suspend

 

Choose this option to suspend a transaction.

Retrieve

 

Choose this option to retrieve a transaction.

Orders

 

Choose this option to add or modify an order.

Find Basket

 

Choose this option to find an item basket and add the items to the transaction.

 

External Order

 

Choose this option to search for an external order and add the order to the transaction for payment.

 

Bill Pay

 

Choose this option to process bill payments.

 

Press Undo to return to the previous screen.
