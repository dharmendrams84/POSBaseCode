Reason Codes

Edit Reason Code

·         To edit a reason code, select a reason code and press F3/Edit.  The Edit Reason Code screen appears. 

Change Default Reason Code

·         To make a reason code the default, select a reason code and press F2/Make Default.   

Note: The reason code order is not changed.

Add Reason Code

·         To add a reason code, press F4/Add.

·         The Add Reason Code screen appears. 

Delete a Reason Code

·         To delete a reason code, select a reason code and press F5/Delete.   

·         When prompted, verify that you want to remove the reason code

 

Change Reason Code Sequence

To change the sequence that the reason codes appear in the list box, you can move items in the list up or down.

·         Press F6/Move Up to move the reason code higher in the list.

·         Press F7/Move Down to move the reason code lower in the list.

Done (Save Changes)

When you are finished making all the changes you desire, press F8/Done.  The changes are saved to the database.
