What Can I Do On This Screen?

 

Enter the number of the transaction you want to post void, then press Next.

 

Press Back to return to the previous screen.

Facts About Post Voiding Transactions

 

The following transactions can be post voided:

 

-  Sale

 

-  Return

 

-  Pay in

 

-  Cash disbursement

 

-  Cash in and cash out

 

Transactions must be post voided on the same business day and on the same register they are originally entered. 

 

A completed transaction that has been modified cannot be post voided. For example, a sale transaction that later has an item returned is considered “modified,” so it can no longer be post voided.

 

Any transaction that has not been completed or tendered can simply be cancelled (press Cancel). To reverse a completed or tendered transaction, the transaction must be post voided.

 

A completed brand credit card enrollment cannot be post voided. Brand credit card enrollment only can be cancelled during the enrollment process before the system dials out for authorization.

 

Only transactions that are assigned transaction numbers can be post voided.

 

When post voiding a gift certificate, you are prompted to frank the gift certificate with void information.

 

If you void or post void a merchandise certificate after the merchandise certificate has already been franked as issued, you are prompted to frank the merchandise certificate with void information.

 

You cannot reprint a receipt for a transaction that has been voided or post voided.

 

If a transaction that contains a credit card tender is post voided, the system prints a duplicate copy of the post void for the customer’s records.

 

A transaction that contains an external order or a gift card issue, reload, or redeem cannot be post voided.
