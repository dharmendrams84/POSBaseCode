Select ID Type

 

Use this screen to choose the customer’s valid form of identification when accepting a check as tender or if the transaction involves PAT cash.

 

What Can I Do On This Screen?

 

Choose the type of identification from the ID Type drop-down menu, then press Next.

 

Note: Valid forms of ID include driver’s license, state identification card, military identification, student identification, and passport.

 

Press Undo to return to the previous screen.

 

Press Cancel to stop the tendering function and return to the Sell Item screen.
