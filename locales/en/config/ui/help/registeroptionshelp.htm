Register Open

If the register is not open, perform these steps to open the register:

·         From the Main Options screen, press F2/Daily Ops. The Daily Ops screen appears.

·         Type your user ID and press Enter/Next.

·         Type your Password and press Enter/Next. 

·         If the Store is not already open, press F2/Start of Day to open the store.  For more information, go to Start of Day.

·         Press F3/Register Open to access the Identification screen. 

Note:   If the Register Already Open Notice appears, it means the register has already been opened by someone else, or it was not closed properly.  Click OK to continue.

·         The Register Open Successful Notice appears. 

·         Press Enter to continue.

Register Close

If the register is open, perform these steps to close it:

·         From the Main Options screen, press F2/Daily Ops. The Daily Ops screen appears.

·         Type your user ID and press Enter/Next.

·         If prompted, type your password and press Enter/Next. 

·         The Daily Ops. Screen appears.

·         Press F5/Register Close. 

Note:   If the till is not closed, the Till Not Closed Notice appears.  Press No to close the till, or Yes to return to the Daily Operations menu.

·         The Register Close Successful Notice appears.  Press OK to continue.

·         You return to the Daily Ops screen.
