Suspend

If a sale, exchange, return, layaway, or special order is interrupted for any reason and you do not want to cancel it, you can suspend the transaction for as long as the till is open.  A transaction can be suspended at any time before tender is completed. 

 

To suspend a transaction, from the Sell Item screen perform these steps:

·         Press F5/Transaction.  The Trans. Options screen appears. 

·         Press F9/More and F2/Suspend   The Suspend Reason screen appears. 

·         You are prompted for a Reason Code.  Choose one from the list and press Enter/Next. 

·         The transaction is suspended and a message is displayed indicating the transaction has been suspended.

·         Press Enter/Next to continue.  The suspended transaction receipt is printed, and you return to the Sell Item Screen.
