Item Options

 

Use this screen to select an item option.

 

What Can I Do On This Screen?

 

Choose an option from the navigation bar on the right side of the screen:

 

Inquiry

Choose this option to inquire on an item.

 

Quantity

Choose this option to enter the quantity of the item.

 

Sales Associate

 

Choose this option to assign a sales associate to the item.

Gift Registry

 

Choose this option to select a gift registry number.

Tax

 

Choose this option to select a tax option for the item. This option is not available when VAT is enabled.

 

Services

 

Choose this option to select a non-merchandise item.

Serial Number

 

Choose this option to assign a serial number to the item.

Gift Receipt

 

Choose this option to print a gift receipt for the item.

Send

 

Choose this option to send the item.

Components

 

Choose this option to view the components of a kit item.

Alternations

 

Choose this option to view the details of an alteration type item.

Pickup

 

Choose this option to set up the item for pickup.

Delivery

 

Choose this option to set up the item for delivery.

 

 

Press Undo to return to the previous screen.
