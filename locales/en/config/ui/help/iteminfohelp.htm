Item Display

 

Information about an item is displayed. Use this screen to search for another item, add an item to the transaction, search for inventory information on an item, or create a label for the item.

 

What Can I Do On This Screen?

 

Choose an option from the navigation bar on the right side of the screen:

 

Search

Enter an item number and choose this option to search for the item.

 

Add

Choose this option to add the item to the transaction.

 

Inventory Inquiry

Choose this option to request inventory information on the item.

 

Request Ticket

Choose this option to request that a label be created for the item.

 

 

 

Press Undo to return to the previous screen.

 

Press Cancel to exit the function.

 

Press Delete to clear the current field entry.
