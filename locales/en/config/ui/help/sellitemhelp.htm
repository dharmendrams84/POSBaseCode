Sell Item

 

Use this screen to select options for the transaction.

 

What Can I Do On This Screen?

 

Enter an item number, then press Next.

 

Choose an option from the navigation bar on the right side of the screen:

 

Return

Choose this option to return an item.

 

Item Lookup

Choose this option to look up information for an item.

 

Item

 

Choose this option to modify the item.

Transaction

 

Choose this option to view transaction options.

Tender

 

Choose this option to view tender options.

Customer

 

Choose this option to view customer options.

Pricing

 

Choose this option to view pricing options.

No Sale

 

Choose this option when there is no sale.

House Account

 

Choose this option to view house account options.

Reprint Receipt

 

Choose this option to view reprint options.

Till Functions

 

Choose this option to view till functions.

Manager

 

Choose this option to view manager options.

Gift Card/Cert

 

Choose this option to view gift options.

Redeem

 

Choose this option to view redeem options.

Scan Sheet

 

Choose this option to view the scan sheet.

 

 

Press Delete to clear the current entry field.

 

Press Cancel to exit the function.
