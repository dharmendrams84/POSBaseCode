Search by Employee ID

To find a customer using the employee ID, press F3/Emp. ID. 

·         Enter the ID in the Employee ID field and press Enter/Next.

·         If a single customer match is found, it is displayed.  If several possible matches are found, they are all displayed.  Use the arrow up and down keys to select the desired customer and press Enter/Next. 

·         The Customer Master screen appears.  Make changes if desired and press Enter/Next.

Note:   In order for employee records to be located via this search, they must already exist.  You cannot add employee records.
