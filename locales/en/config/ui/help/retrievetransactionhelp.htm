Retrieve

A suspended transaction can be retrieved from any register/terminal in the store if no other transaction is active on that register.  If a transaction is active, it must be suspended or completed before you can retrieve the desired transaction.

 

To retrieve a transaction from the Sell Item screen, perform these steps:

·         Press F5/Transaction.  The Trans. Options screen appears.  Press F9/More>>> and F3/Retrieve. 

·         The Enter Transaction screen appears.  You can scan or type the transaction and press Enter/Next, or you can press F2/List to see a list of suspended transactions.

·         If you selected F2/List, the Suspend List screen appears. 

·         If more than one transaction is listed, select the transaction you want to retrieve from the list and press Enter/Next. 

·         The suspended transaction is retrieved and the transaction details are displayed.

·         You can complete, change, or cancel the transaction:

–                    To complete the transaction, press F6/Tender and proceed normally.

–                    To change the transaction, make the desired changes, press F6/Tender, and proceed normally.

–                    To cancel the transaction, go to Cancel Suspended Transaction.

Cancel Suspended Transaction

All suspended transactions must be resolved before closing the till each day.  These transactions must be retrieved, and either completed or cancelled.  To cancel a suspended transaction, perform these steps:

·         Press F5/Transaction.  The Trans. Options screen appears.  Press F9/More>>> and F3/Retrieve. 

·         The Enter Transaction screen appears.  You can scan or type the transaction and press Enter/Next or you can press F2/List to see a list of suspended transactions.

·         If you selected F2/List, the Suspend List screen appears. 

·         If more than one transaction is listed, select the transaction you want to retrieve from the list and press Enter/Next. 

·         Select the transaction you want to cancel from the list using the arrow keys, and press F11/Clear. 

·         You are prompted to verify you want to cancel the transaction.  Press Yes to confirm and cancel the suspended transaction. 

·         You return to the Sell Item Screen.
