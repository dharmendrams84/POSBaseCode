Gift Receipt

A customer may request a gift receipt to provide with a gift with no price shown.  The person receiving the gift can return the item using the gift receipt.  To request a gift receipt, perform these steps:

·         From the Sell Item screen, select the item and press F4/Item.  The Item Options Screen appears.  If you need to select multiple items, select the item and press the spacebar until you select all the desired items.

·         Press F9/More>>> to display page two of the Item Options screen.

·         When the Item Options More screen appears, press F5/Gift Receipt. 

·         The Sell Item screen reappears.  When the transaction is completed, a gift receipt prints with the other receipt(s) for each selected item.
